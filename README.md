# Rental Verification System

A modern full-stack rental verification web application built using **React** for the frontend and **Node.js** for the backend. This project demonstrates clean architecture, proper separation of frontend and backend concerns, and practical full-stack development suitable for portfolio and real-world extension.

---

## Overview

The Rental Verification System is designed to validate and manage rental or tenant-related information through a structured web interface. It can be extended for use in property management systems, tenant screening platforms, or verification workflows.

This repository is intended for **learning, demonstration, and portfolio purposes**.
